Como importar múltiplos recibos?

Para importar vários recibos de uma vez, prepare uma planilha Excel (.xlsx ou .xls). Escolha o modelo adequado ao tipo de recibo que você está usando:

Modelo Padrão

A B C D E F G
1 Valor Recebedor Referencia Descricao Emitente CPFCNPJ Data
2 R$ 1.500,00 João Silva Serviços de consultoria Projeto X - Fase 1 Maria Santos 123.456.789-00 15/01/2024
Dica Padrão: Use as 7 colunas do modelo padrão para recibos tradicionais com informações detalhadas.

Modelo Completo

A B C D E F G H
1 NumeroRecibo Valor Recebedor Referencia Descricao Emitente CPFCNPJ Data
2 0001 R$ 1.500,00 João Silva Serviços de consultoria Projeto X - Fase 1 Maria Santos 123.456.789-00 15/01/2024
Dica Completo: Use a coluna NumeroRecibo para numerar cada documento e manter a sequência organizada.

Modelo Simplificado (Básico)

A B
1 Valor Recebedor
2 R$ 850,00 Ana Costa
3 R$ 1.200,00 Pedro Santos
Dica Simplificado: Use apenas 2 colunas para recibos rápidos e simples. Perfeito para uso pessoal e situações informais!
Importante: Os nomes das colunas devem ser exatamente como mostrado. O sistema aceita datas e valores em qualquer formato!
